# Nour — Next Iteration Changelog

## Added
- V1 core tracker registry with three active default trackers: Mood, Energy, Self-talk.
- Internal tracker registry preserves inactive trackers for future configuration/theming work.
- App tagline added: “Healing Has Curves. Follow trends, not moments.”
- Baseline-relative helper copy added to slider pages.
- Notes page clarified as optional, with no skip action.
- Notes now use a soft internal max length of 2,000 characters without a visible counter.
- Notes export inclusion is user-controlled and defaults off for privacy.
- Chart paths now use softer curved rendering.
- Export chart keeps a cleaner, more professional visual style than the in-app chart.

## Removed / Hidden
- Removed live/dynamic slider category labels such as “cruel,” “harsh,” etc.
- Removed feelings/events tag steps from the V1 check-in flow.
- Removed feelings/events tag visualization from the V1 home screen.
- Reduced check-in flow from six sliders plus tags/notes to three sliders plus optional notes.

## Updated
- Slider pages now use a more fixed layout rhythm.
- Skip action moved beneath the primary Next action on slider pages.
- Trend/composite calculation now uses only active V1 trackers.
- Existing inactive tracker data is preserved when editing old entries.
- Export notes copy updated to reflect that notes are included only when explicitly enabled.

## Deferred / Preserved for V2+
- Tracker configuration UI.
- Feelings/events tags UI and analytics.
- Missed/skipped day chart toggle UI.
- Retroactive fill-in behavior.
- Visual theming/style systems.
- Advanced therapist/export modes.
